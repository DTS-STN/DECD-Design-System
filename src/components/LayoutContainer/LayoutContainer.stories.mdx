# LayoutContainer 

This utility container helps layout content across screen sizes according to our TailwindCSS implementation of Bootstrap 5 breakpoints and maximum container widths.

You can use this container to wrap a page, section or component or you could simiply add the 'container' class to the top-most html element on your page, section or component.
